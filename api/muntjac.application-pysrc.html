  46      """Base class required for all Muntjac applications. This class provides 
  47      all the basic services required by Muntjac. These services allow external 
  48      discovery and manipulation of the user, L{com.vaadin.ui.Window windows} and 
  49      themes, and starting and stopping the application. 
  50   
  51      As mentioned, all Muntjac applications must inherit this class. However, 
  52      this is almost all of what one needs to do to create a fully functional 
  53      application. The only thing a class inheriting the C{Application} 
  54      needs to do is implement the C{init} method where it creates the 
  55      windows it needs to perform its function. Note that all applications must 
  56      have at least one window: the main window. The first unnamed window 
  57      constructed by an application automatically becomes the main window which 
  58      behaves just like other windows with one exception: when accessing windows 
  59      using URLs the main window corresponds to the application URL whereas other 
  60      windows correspond to a URL gotten by catenating the window's name to the 
  61      application URL. 
  62   
  63      See the class C{muntjac.demo.HelloWorld} for a simple example of 
  64      a fully working application. 
  65   
  66      B{Window access.} C{Application} provides methods to 
  67      list, add and remove the windows it contains. 
  68   
  69      B{Execution control.} This class includes method to start and finish the 
  70      execution of the application. Being finished means basically that no 
  71      windows will be available from the application anymore. 
  72   
  73      B{Theme selection.} The theme selection process allows a theme to be 
  74      specified at three different levels. When a window's theme needs to be 
  75      found out, the window itself is queried for a preferred theme. If the 
  76      window does not prefer a specific theme, the application containing the 
  77      window is queried. If neither the application prefers a theme, the default 
  78      theme for the L{terminal<muntjac.terminal.terminal.ITerminal>} is used. The 
  79      terminal always defines a default theme.
