                                                             Accordion

An accordion is a component similar to a TabSheet, but with a vertical orientation and the selected component presented between tabs.

Closable tabs are not supported by the accordion.

The Accordion can be styled with the .v-accordion, .v-accordion-item, .v-accordion-item-first and .v-accordion-item-caption styles.

See Also: TabSheet
